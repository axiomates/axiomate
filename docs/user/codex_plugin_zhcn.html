Codex 插件安装与使用

本文说明如何在 Axiomate 中安装、刷新和初始化 Codex 插件。

用途

Codex 插件把 OpenAI Codex CLI 接入 Axiomate，用于代码审查、后台任务、救援式调查和可选的停止前 review gate。

插件 marketplace 名称是 openai-codex，插件名称是 codex，因此安装目标写作 codex@openai-codex。

首次安装

先添加 marketplace，再安装插件并重新加载：

/plugin marketplace add axiomates/codex-plugin-cc
/plugin install codex@openai-codex
/reload-plugins
/codex:setup

/codex:setup 会检查本机 Node.js、npm、Codex CLI 和 Codex 认证状态。如果提示未登录，按输出中的提示运行 !codex login。

重新安装或刷新

如果之前安装过旧版本，推荐先卸载再安装：

/plugin uninstall codex
/plugin install codex@openai-codex
/reload-plugins
/codex:setup

如果只是想拉取 marketplace 上的新版本，也可以先刷新 marketplace，再更新插件：

/plugin marketplace refresh
/plugin update codex
/reload-plugins
/codex:setup

常用命令

  • /codex:setup：检查 Codex CLI 是否可用，并显示下一步操作。
  • /codex:status：查看当前 Codex runtime、review gate 状态和后台任务。
  • /codex:review：对当前 git 工作区或分支运行 Codex 代码审查。
  • /codex:adversarial-review：运行更偏设计质疑和假设挑战的审查。
  • /codex:rescue：把调查、修复或后续救援任务委托给 Codex。
  • /codex:result：查看已完成 Codex 后台任务的结果。
  • /codex:cancel：取消正在运行的 Codex 后台任务。

可选：启用停止前 review gate

如果希望每次停止会话前由 Codex 做一次审查，可以启用 review gate：

/codex:setup --enable-review-gate

关闭 review gate：

/codex:setup --disable-review-gate

注意：review gate 会在停止阶段运行 Codex 审查，可能增加等待时间。建议确认 Codex CLI 和认证状态正常后再启用。
